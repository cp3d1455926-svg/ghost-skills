# ✈️ Travel Planner - 旅行规划助手

## 触发规则

**关键词：**
- 旅行规划、旅游计划、行程安排
- 去哪里玩、推荐景点、旅游攻略
- 预算估算、旅行费用

**场景：**
- 用户计划旅行需要行程建议
- 用户查询目的地景点
- 用户需要预算估算

## 功能描述

### 核心功能

1. **行程规划**
   - 根据天数规划行程
   - 景点路线优化
   - 时间分配建议

2. **景点推荐**
   - 热门景点推荐
   - 小众景点挖掘
   - 季节性推荐

3. **预算估算**
   - 交通费用
   - 住宿费用
   - 餐饮费用
   - 门票费用

4. **旅行贴士**
   - 最佳旅行时间
   - 必备物品清单
   - 注意事项

## 权限需求

- [x] 网络访问（景点/酒店查询）
- [x] 文件读写（行程存储）
- [ ] 外部 API（地图/旅游平台）

## 示例对话

```
用户：帮我规划一个 3 天的北京旅行
助手：好的！帮你规划北京 3 日游：

📅 **Day 1 - 经典景点**
- 上午：天安门广场 → 故宫
- 下午：景山公园 → 北海公园
- 晚上：王府井步行街

📅 **Day 2 - 文化之旅**
- 上午：颐和园
- 下午：圆明园
- 晚上：中关村/五道口

📅 **Day 3 - 长城之旅**
- 全天：八达岭长城
- 晚上：返回市区

💰 **预算估算**（人均）
- 住宿：300 元/晚 × 2 = 600 元
- 餐饮：150 元/天 × 3 = 450 元
- 交通：200 元
- 门票：500 元
- 总计：约 1750 元

需要我帮你预订酒店或查询详细交通吗？
```
