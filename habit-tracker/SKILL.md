# 🎯 Habit Tracker - 习惯养成助手

## 触发规则

**关键词：**
- 习惯打卡、每日打卡、习惯追踪
- 坚持、养成习惯
- 统计、习惯报告

**场景：**
- 用户想养成新习惯
- 用户需要每日打卡提醒
- 用户想查看习惯统计

## 功能描述

### 核心功能

1. **习惯创建**
   - 设定习惯目标
   - 设置提醒时间
   - 设定频率（每日/每周）

2. **打卡系统**
   - 每日打卡
   - 补打卡
   - 打卡提醒

3. **统计分析**
   - 连续天数
   - 完成率
   - 趋势图表

4. **激励机制**
   - 成就徽章
   - 里程碑奖励
   - 打卡 streak

## 权限需求

- [x] 文件读写（习惯数据存储）
- [x] 定时任务（提醒）

用户：打卡读书
