# 🎯 Goal Manager - 目标管理助手

## 触发规则

**关键词：**
- 目标设定、OKR、KPI
- 目标跟踪、进度管理
- 复盘、总结

**场景：**
- 用户想设定年度/月度目标
- 用户需要跟踪目标进度
- 用户想做目标复盘

## 功能描述

### 核心功能

1. **目标设定**
   - SMART 原则指导
   - OKR 框架
   - 目标分解

2. **进度跟踪**
   - 定期更新进度
   - 里程碑标记
   - 进度可视化

3. **复盘总结**
   - 周/月/年复盘
   - 成功/失败分析
   - 经验总结

4. **提醒督促**
   - 进度提醒
   - 截止日期提醒
   - 激励话语

## 权限需求

- [x] 文件读写（目标数据存储）
- [x] 定时任务（提醒）

## 示例对话

```
用户：帮我设定 2026 年的目标
助手：好的！帮你设定年度目标：

🎯 **2026 年度目标**

**O1: 职业发展**
- KR1: 完成 3 个重要项目
- KR2: 学习 2 项新技能
- KR3: 晋升到高级工程师

**O2: 健康生活**
- KR1: 每周运动 3 次
- KR2: 体重控制在 70kg
- KR3: 每天睡眠 7 小时

**O3: 个人成长**
- KR1: 阅读 24 本书
- KR2: 学习一门新语言
- KR3: 旅行 3 个城市

需要我帮你分解为月度目标吗？

用户：更新目标进度
助手：好的！请告诉我哪个目标的进度：

🎯 职业发展 - 完成 3 个重要项目
📊 当前进度：1/3 (33%)
💬 备注：第一个项目已完成

✅ 已更新！继续加油！

用户：做月度复盘
助手：📋 **3 月复盘报告**

✅ **完成的目标**
- 阅读 2 本书 📚
- 运动 12 次 🏃

⏳ **进行中的目标**
- 学习 Python (50%)
- 减肥计划 (30%)

❌ **未完成的目标**
- 早起打卡 (完成率 40%)

💡 **改进建议**
- 设置更早的睡觉提醒
- 找运动伙伴互相督促
```
