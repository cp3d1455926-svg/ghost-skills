# 📊 PPT Generator - PPT 生成助手

## 触发规则

**关键词：**
- 做 PPT、生成演示文稿、幻灯片
- PPT 大纲、PPT 模板
- 配图、排版

**场景：**
- 用户需要根据内容生成 PPT
- 用户需要 PPT 大纲
- 用户需要配图建议

## 功能描述

### 核心功能

1. **PPT 大纲生成**
   - 根据主题生成结构
   - 页数建议
   - 内容分配

2. **PPT 内容生成**
   - 每页标题
   - 要点列表
   - 演讲备注

3. **智能配图**
   - 根据内容推荐图片
   - 无版权图片搜索
   - 图标推荐

4. **模板推荐**
   - 商务风格
   - 学术风格
   - 创意风格

## 权限需求

- [x] 文件读写（PPT 导出）
- [x] 网络访问（图片搜索）
- [ ] 外部 API（PPT 生成服务）

## 示例对话

```
用户：帮我做一个产品介绍 PPT
助手：好的！帮你生成产品介绍 PPT 大纲：

📊 **PPT 大纲** - 产品介绍
📄 建议页数：12 页

**Slide 1: 封面**
- 标题：[产品名称]
- 副标题：[产品 Slogan]
- 配图：产品主图

**Slide 2: 目录**
- 产品概述
- 核心功能
- 技术优势
- 应用场景
- 客户案例
- 价格方案

**Slide 3: 产品概述**
- 产品定位
- 目标用户
- 核心价值
- 配图：产品架构图

...

需要我：
1. 生成完整内容
2. 推荐配图
3. 导出为 PPT 文件
```
